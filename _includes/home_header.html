When we run over libraries, persuaded of these principles, what havoc must we make? If we take in our hand any volume; [...] let us ask, Does it contain any abstract reasoning concerning quantity or number? No. Does it contain any experimental reasoning concerning matter of fact and existence? No. Commit it then to the flames: for it can contain nothing but sophistry and illusion. David Hume, An Enquiry Concerning Human Understanding, 1777
